Using Automatic Wireless (one-push) Setup
The Automatic Wireless Setup option allows this printer/multifunction device to receive its wireless network configuration settings directly from any wireless access point that supports Wi-Fi Protected Setup or AOSSTM.
If you encounter a problem during setup, temporarily place your machine closer to the wireless access point.
For further information on how to configure your wireless access point to work in Wi-Fi Protected Setup and AOSSTM modes refer to the manufactures user guide.
Once the printer/multifunction device is configured you can move it back to a convenient position within your working environment that is within range of your wireless access point.
