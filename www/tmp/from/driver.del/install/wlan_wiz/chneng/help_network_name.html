Wireless Network Name
Enter the network name (SSID) that the device will be associated with.

If the device is communicating in an Ad-hoc network and there is no access point, enable the "This is an Ad-hoc network and there is no access point" option. If you select this option you must also manually select the network channel you are using. If you do not know which network channel you are using refer to the documentation supplied with your computer and look for information on channel settings.
