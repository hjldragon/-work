IP Address Configuration
The wireless device must be configured to use an IP address. There are two possible options:

Obtain an IP address automatically. Use this option if your network is using a DHCP server.

Set Static IP address. Use this option if your network is not using a DHCP server and you wish to manually enter the IP address and subnet mask settings.

If you are not sure which option to select, please speak to your system administrator.

Please note: if you select the "Obtain an IP address automatically" option, your device will attempt to acquire an IP address from a DHCP server. If it is not able to communicate with the DHCP server because the DHCP server is unavailable or you in-correctly configured your wireless security settings, the device will switch to an APIPA based IP address in the 169.254.0.1 to 169.254.255.254 range.
