IP地址配置
无线设备必须配置为使用一个IP地址。有两个可能的选项：

自动获得一个IP地址。如果您的网络正在使用一台DHCP服务器，请使用这种选项。

设定静态的IP地址。如果您的网络没有使用DHCP服务器并且您希望手动输入IP地址和子网掩码设置，请使用这种选项。

如果您不能确定选择哪一种，请联系您的系统管理员

请注意：如果您选择了“自动获得IP地址”选项，您的设备将尝试从一台DHCP服务器上获得一个IP地址。如果由于DHCP服务器无效或者您没有正确配置无线安全设置而使它不能与DHCP服务器通信，设备将在169.254.0.1到169.254.255.255的范围内转向基于APIPA的一个IP地址。
