网络密钥配置
输入正确的网络密钥并单击“下一步”。每种安全方法都有其些规则。
使用 WEP 打开系统/共享密钥
此密钥为必须以 ASCII 或十六进制格式输入的 64 位或 128 位值。
  • 64(40)位 ASCII：
    使用 5 个文本字符，如“WSLAN”（区分大小写）。
  • 64(40)位十六进制：
    使用 10 位数的十六进制数据，如“71f2234aba”。
  • 128(104)位 ASCII：
    使用 13 个文本字符，如“Wirelesscomms”（区分大小写）。
  • 128(104)位十六进制：
    使用 26 位数的十六进制数据，如“71f2234ab56cd709e5412aa2ba”。
WPA-PSK/WPA2-PSK 和 TKIP 或 AES
使用最少 8 个字符，最多 63 个字符长度的预共享密钥（PSK）。
