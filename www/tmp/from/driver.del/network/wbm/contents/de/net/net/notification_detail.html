Benachrichtigung
Administratoradresse
Sie können die E-Mail-Adresse eines Administrators eingeben, der dann die Statusinformation per E-Mail erhält.
Es können 2 Administratoren registriert werden.
Eine E-Mail schicken, wenn das Problem behoben wurde.
Wählen Sie diese Option um eine E-Mail zu erhalten, die Sie benachrichtigt, dass das Problem behoben wurde.
Fehlerart
Sie können entscheiden, wann eine E-Mail für einen bestimmten Fehler geschickt wird, indem Sie 'xx Minuten' und 'x Mal' einstellen. Eine E-Mail wird 'xx Minuten', nachdem der entsprechende Fehler aufgetreten ist, geschickt. Eine E-Mail wird geschickt, wenn der entsprechende Fehler 'xx Mal' aufgetreten ist.
