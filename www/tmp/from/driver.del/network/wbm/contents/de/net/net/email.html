Auf dieser Seite können die Parameter für das Gerät geändert werden.
E-Mail-Einstellungen senden (SMTP)
Serveradresse
In diesem Feld wird der Hostname oder die IP-Adresse eines SMTP-Mailservers (Server für ausgehende E-Mails) in Ihrem Netzwerk angezeigt (z. B. smtp.beispiel.net oder 192.0.0.1).
Wenn Sie den DNS-Namen des SMTP-Servers verwenden, müssen Sie den vollständigen Domänennamen (FQDN = Fully Qualified Domain Name) wie z. B. smtp.beispiel.net angeben.
Port
Geben Sie die Nummer des SMTP-Server-Ports an, zu dem die Verbindung hergestellt werden soll.
Serverauthentifizierungsmethode
Geben Sie die Authentifizierungsmethode für den SMTP-Server an. Dieser Wert bestimmt die Methode, die den Benutzer beim SMTP-Server authentifiziert.
Keine: Keine Authentifizierung beim SMTP-Server.
SMTP-AUTH : Nutzt die Methode "SMTP-AUTH" (SMTP-Authentifizierung). Bei dieser Methode müssen die SMTP-Kontodaten richtig gesetzt sein.
Name für SMTP-AUTH-Konto
Kennwort für SMTP-AUTH-Konto
Geben Sie zum Validieren beim SMTP-Server Namen und Kennwort an. Wenn Sie kein Kennwort nutzen möchten, dann geben Sie bitte eine Leerstelle ein.
E-Mail-Adresse des Geräts
Geben Sie die E-Mail-Adresse des Geräts ein. Wurde die Administratoradresse nicht korrekt eingegeben, so wird die Benachrichtigung zum Gerät gesendet. Wird keine E-Mail-Adresse für das Gerät angegeben, so erfolgt keine Benachrichtigung an den Administrator.
