TCP/IP (Câblé)
Généralités
Cette page permet de modifier les paramètres TCP/IP.
REMARQUE :
si vous désactivez le Protocole Internet (IP), vous ne pourrez pas configurer l'appareil à l'aide du web.
Adresse IP
Masque de sous-réseau
Entrez les paramètres de l'adresse IP à assigner à l'appareil (demandez à votre administrateur réseau quelle est l'adresse IP à utiliser).
Passerelle
Si vous utilisez une passerelle (routeur), entrez son adresse ici (demandez quelle est l'adresse de passerelle à utiliser à votre administrateur réseau).
Méthode d'amorçage
Si la méthode est STATIC, les valeurs présentes dans les champs d'adresse IP, sous-réseaux et passerelle sont utilisées par l'appareil. Sinon, ce champ indique quel sera le protocole de réseau utilisé pour acquérir ces paramètres quand le serveur est allumé ou réinitialisé. Si vous activez le protocole APIPA, l'appareil utilisera une adresse IP générée par APIPA si un serveur DHCP ou un autre serveur capable d'attribuer des adresses IP automatiques n'est pas disponible.
