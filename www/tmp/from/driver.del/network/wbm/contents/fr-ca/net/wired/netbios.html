NetBIOS (Câblé)
Généralités
Cette page permet de modifier les paramètres NetBIOS.
NETBIOS/IP
Permet les communications à l'aide du protocole Internet (IP)
Nom de l'ordinateur
Le nom de l'ordinateur correspond aux 15 premiers caractères du nom de nœud de l'appareil.
Méthode du serveur WINS
Pour NetBIOS/IP, avec STATIC, l'appareil utilise les valeurs présentes dans les champs serveur WINS. Avec AUTO, l'appareil essaie d'utiliser DHCP pour acquérir ces paramètres quand le serveur est mis sous tension ou réinitialisé. La méthode BOOT sur la page TCP/IP doit être configurée sur Auto ou DHCP pour activer DHCP. Pour désactiver l'enregistrement WINS, configurez la méthode sur STATIC et fixez les adresses IP des serveurs WINS primaire et secondaire à 0.0.0.0
Adresse IP du serveur WINS primaire
Adresse IP du serveur WINS secondaire
Entrez l'adresse IP voulue du serveur WINS primaire, et du serveur WINS secondaire optionnel à utiliser par l'unité.
