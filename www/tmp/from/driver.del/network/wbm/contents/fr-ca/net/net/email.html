SMTP
Généralités
Cette page permet de modifier les paramètres de l'appareil.
Envoyer les paramètres du courriel(SMTP)
Adresse du serveur
Le champ affiche le nom d'hôte ou l'adresse IP d'un serveur de messagerie SMTP (Serveur d'envoi de courriels) sur votre réseau. (Ex : smtp.exemple.net ou 192.0.0.1).
Si vous utilisez le nom DNS du serveur SMTP, vous devez utiliser le FDQN (Fully Qualified Domain Name) tel que smtp.exemple.net.
Port
Spécifiez le numéro du port de destination du serveur SMTP auquel vous allez vous connecter.
Méthode d'authentification du serveur
Spécifiez la méthode d'authentification pour le serveur SMTP. Cette valeur détermine le type de méthode utilisé pour authentifier l'utilisateur auprès du serveur SMTP.
Aucun : ne s'authentifie pas auprès du serveur SMTP.
SMTP-AUTH: utilise la méthode qui correspond à l'authentification SMTP. Pour cette méthode, il est nécessaire de définir correctement les détails du compte SMTP.
Nom du compte SMTP-AUTH
Mot de passe du compte SMTP-AUTH
Spécifiez un nom et un mot de passe à valider avec le serveur SMTP. Pour spécifier qu'aucun mot de passe n'est requis, entrez un simple espace.
Adresse de courriel de l'appareil
Entrez l'adresse de courriel de l'appareil. Si l'adresse d'un administrateur n'a pas été entrée correctement, le courriel d'avis sera envoyé à l'appareil. Si vous ne définissez pas l'adresse de courriel de l'appareil, le courriel d'avis n'est pas envoyé à l'administrateur.
