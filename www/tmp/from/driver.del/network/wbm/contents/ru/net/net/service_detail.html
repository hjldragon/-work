Служба
Обзор
На этой странице можно изменить параметры, связанные со службой, выбранной в списке служб.
Имя службы
Указывается имя службы. Обычно изменить значение в этом поле не нужно.
Порт службы
Указывается связанный со службой физический порт.
Протоколы
Эти флажки определяют разрешенные для этой службы сетевые протоколы.
Фильтр
Здесь можно выбрать фильтр данных для службы.
0 - нет фильтра, данные проходят без изменения.
1 - подстановка текста, по умолчанию CRLF для LF.
2 - означает задание AppleTalk. Фильтрация зависит от оборудования и выбранных параметров.
3 - обычный текст преобразуется в Postscript.
4 - выходные данные преобразуются в Postscript Tagged Binary.
Управляющие строки
Указывает строки данных, отправляемые в начале или в конце задания.
Порт Raw TCP
Этот параметр определяет порт TCP (если не равен нулю), используемый для подключения к данной службе.
Двусторонняя связь
Если этот флажок установлен, служба отправит данные обратно с устройства в сеть. Обычно менять значение этого параметра не следует. В случае недвунаправленного устройства PostScript можно отключить поддержку двунаправленности, чтобы включить спуфинг AppleTalk.
