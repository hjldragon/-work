На этой странице можно изменить параметры устройства.
Настройка отправки эл. почты (SMTP)
Адрес сервера
В этом поле указывается имя или IP-адрес сервера исходящих сообщений электронной почты (SMTP). (Например smtp.example.net или 192.0.0.1)
Если вы используете DNS-имя сервера исходящей почты (SMTP), необходимо использовать FQDN (Fully Qualified Domain Name, полное имя узла), например smtp.example.net.
Порт
Укажите номер порта назначения для сервера SMTP, с которым будет выполнено соединение.
Метод аутентификации сервера
Укажите способ проверки подлинности на сервере SMTP. Это значение определяет способ, который будет использоваться для проверки подлинности пользователя на сервере SMTP.
Нет: проверка подлинности на сервере SMTP не выполняется.
SMTP-AUTH: проверка подлинности на сервере SMTP. Для использования этого способа необходимо правильно настроить параметры работы с сервером SMTP.
Имя учетной записи SMTP-AUTH
Пароль учетной записи SMTP-AUTH
Укажите имя пользователя и пароль, подлинность которых будет проверяться сервером SMTP. Чтобы не использовать пароль, введите один пробел.
Адрес электронной почты устройства
Введите адрес электронной почты устройства. Если адрес администратора был введен неправильно, то уведомление будет отправлено на устройство. Если адрес электронной почты устройства не указан, уведомление администратору не отправляется.
