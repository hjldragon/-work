NETBIOS/IP
Значение 'Включить' разрешает связь по протоколу IP (Internet Protocol).
Имя компьютера
'Имя компьютера' — это первые 15 символов имени узла устройства.
Метод сервера WINS
Для NetBIOS/IP в режиме STATIC устройством используются значения, указанные в полях сервера WINS. В режиме AUTO устройство попытается использовать DHCP для получения этих параметров при запуске или перезагрузке сервера. Для включения DHCP на странице TCP/IP должен быть установлен метод загрузки «Авто» или DHCP. Чтобы отключить регистрацию WINS, установите метод STATIC и задайте для основного и дополнительного WINS-серверов IP-адреса 0.0.0.0
