Panoramica
Questa pagina consente di modificare i parametri NetBIOS.
NETBIOS/IP
Attiva la comunicazione sul protocollo IP.
Nome computer
Il nome del computer riflette i primi 15 caratteri del nome del nodo del dispositivo.
Metodo server WINS
Per NetBIOS/IP, se impostato su STATIC, i valori presenti nei campi del server WINS vengono utilizzati dal dispositivo. Se l'impostazione è AUTO, il server di stampa tenta di utilizzare il protocollo DHCP per acquisire questi parametri all'accensione o al riavvio del server. Metodo BOOT nella pagina TCP/IP deve essere impostato su Auto o DHCP per attivare DHCP. Per disattivare la registrazione WINS, impostare il metodo su STATIC e gli indirizzi IP del server WINS principale e secondario su 0.0.0.0
Indirizzo IP server WINS primario
Indirizzo IP server WINS secondario
Immettere l'indirizzo IP del server WINS primario e secondario opzionale che l'unità dovrà utilizzare.
