SMTP
Panoramica
Questa pagina consente di modificare i parametri del dispositivo.
Invia impostazioni e-mail (SMTP)
Indirizzo server
Questo campo visualizza il nome dell'HOST o l'indirizzo IP del mail server SMTP (outgoing E-mail Server) della rete. (Es. smtp.esempio.net - o - 192.0.0.1)
Se si sta utilizzando il nome DNS dell'E-mail server SMTP occorre utilizzare FQDN (Fully Qualified Domain Name) ad esempio smtp.esempio.net.
Porta
Specificare il numero della porta di destinazione del server SMTP al quale viene effettuata la connessione.
Metodo autenticazione server
Specificare il metodo di autenticazione per il server SMTP. Questo valore determina il tipo di metodo utilizzato per autenticare l'utente al server SMTP.
Nessuno: non viene utilizzata alcuna autenticazione al server SMTP.
SMTP-AUTH: viene utilizzato il metodo "SMTP-AUTH", ovvero l'autenticazione SMTP. Se si utilizza questo metodo, è necessario configurare correttamente i dettagli dell'account SMTP.
Nome account SMTP-AUTH
Password account SMTP-AUTH
Specificare un nome e una password da convalidare con il server SMTP. Per non impostare alcuna password, digitare un singolo spazio.
Indirizzo e-mail del dispositivo
Immettere l'indirizzo e-mail del dispositivo. Se l'indirizzo di un amministratore non è stato immesso correttamente, il messaggio di notifica verrà inviato al dispositivo. Se non si imposta un indirizzo e-mail per il dispositivo, il messaggio di notifica non verrà inviato all'amministratore.
