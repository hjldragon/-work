TCP/IP (Cablata)
Panoramica
Questa pagina consente di modificare i parametri TCP/IP.
NOTA :
Se si disattiva il protocollo IP (protocollo Internet), sarà impossibile configurare il dispositivo mediante il Web.
Indirizzo IP
Maschera di sottorete
Immettere le impostazioni dell'indirizzo IP da assegnare al dispositivo (chiedere all'amministratore della rete quale indirizzo IP utilizzare).
Gateway
Se si utilizza un gateway (router), immettere qui il relativo indirizzo (chiedere all'amministratore della rete quale indirizzo gateway utilizzare).
Metodo di avvio
Se impostato su STATIC, i valori presenti nei campi dell'indirizzo IP, della subnet e del gateway vengono utilizzati dal dispositivo. In caso contrario, questo campo indica quale protocollo di rete verrà utilizzato per acquisire questi parametri all'accensione o al riavvio del server. Se si attiva il protocollo APIPA, il dispositivo utilizzerà un indirizzo IP generato da APIPA se non è disponibile un server DHCP o altro in grado di assegnare automaticamente gli indirizzi IP.
