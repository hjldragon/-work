Ethernet
Overview
This page allows you to control the ethernet link mode. Auto allows the Device automatically to select the link mode. If the connection fails by Auto, please try a static configuration and select the static 100BaseTX or 10BaseT link mode.
NOTE : if you set an incorrect static link mode, you might not be able to communicate with your Device.
