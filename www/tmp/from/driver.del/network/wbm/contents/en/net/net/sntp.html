SNTP
Overview
This page allows you to configure the SNTP settings.
SNTP Server Method
STATIC; the Device uses the value set on this page for SNTP server and port fields. AUTO; the Device will attempt to use DHCP to acquire these parameters when the server powers up or is reset. The BOOT Method on the TCP/IP page must be set to Auto or DHCP to enable DHCP.
Primary SNTP Server Address
This field is to input the host name or IP address of the Primary SNTP server. (Ex. sntp.example.net -or- 192.0.0.1) If you are using the DNS name of the SNTP server you must use the FQDN (Fully Qualified Domain Name) such as sntp.example.net.
Primary SNTP Server Port
Specify the destination port number of the Primary SNTP server that you will connect to.
Secondary SNTP Server Address
This field is to input the host name or IP address of the Secondary SNTP server. It is used as a backup to the Primary SNTP server address. If the Primary server is unavailable, the Device will contact the Secondary SNTP server.
Secondary SNTP Server Port
Specify the destination port number of the Secondary SNTP server that you will connect to.
Synchronization Interval
The Device will synchronize with the SNTP server automatically at intervals of specified hours.
Synchronization Status
This field displays the synchronization status with the specified SNTP server.
Succeeded: The last synchronization has succeeded.
Failed to Synchronize: The last synchronization has failed.
SNTP Unspecified: Because of incorrect SNTP settings, synchronization has never succeeded from when the Device powered up. Or the SNTP is disabled.
Synchronizing: The Device is synchronizing with the SNTP server. After a while, press the Refresh button to confirm whether synchronization succeeded.
NOTE:
The Synchronization Status 'Synchronizing' will continue for up to 2 minutes. If the status is 'Failed to Synchronize' or 'SNTP Unspecified', then the specified SNTP server cannot be found. The SNTP server doesn't exist or the specified host doesn't support SNTP. Check the SNTP server name and port number.
