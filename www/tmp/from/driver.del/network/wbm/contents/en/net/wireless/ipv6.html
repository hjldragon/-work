IPv6 (Wireless)
Overview
This page allows you to modify the IPv6 parameters.
By default, IPv6 is disabled. To enable it, click the "Enable" option.
Prioritize IPv6 Address
When this option is selected IPv6 addresses are given priority to IPv4 addresses.
Static IPv6 Address
Enter the IPv6 address you wish to assign to the Device.
Enable this Address
To use the manually entered IPv6 address, select the "Enable this Address" option.
Primary DNS Server IP Address
Secondary DNS Server IP Address
Enter the desired IP address of the primary and optional secondary DNS server.
IPv6 Address List
The list of IPv6 addresses assigned to this machine.
