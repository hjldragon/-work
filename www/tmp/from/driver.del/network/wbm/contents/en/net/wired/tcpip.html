TCP/IP (Wired)
Overview
This page allows you to modify the TCP/IP parameters.
NOTE:
If you disable the IP (Internet Protocol), you will be unable to configure the Device using the web.
IP Address
Subnet Mask
Enter the IP address settings you wish to assign to the Device (check with your network administrator for the IP address to use).
Gateway
If you are using a gateway (router), enter its address here (check with your network administrator for the gateway address to use).
Boot Method
If STATIC, the values present in the IP address, subnet, and gateway fields are used by the Device. If not this field indicates what network protocol will be used to acquire these parameters when the server powers up or is reset. If you enable the APIPA protocol, the Device will use an APIPA generated IP address if a DHCP or other server capable of assigning automatic IP addresses is unavailable.
