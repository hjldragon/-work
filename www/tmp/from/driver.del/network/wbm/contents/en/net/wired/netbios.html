NetBIOS (Wired)
Overview
This page allows you to modify the NetBIOS parameters.
NETBIOS/IP
Enables communication over the Internet Protocol (IP)
Computer Name
The Computer name is the first 15 characters of the Device node name.
WINS Server Method
For NetBIOS/IP, if STATIC, the values present in the WINS server fields are used by the Device. If AUTO, the Device will attempt to use DHCP to acquire these parameters when the server powers up or is reset. The BOOT Method on the TCP/IP page must be set to Auto or DHCP to enable DHCP. To disable WINS registration, set the method to STATIC and set the primary and secondary WINS server IP addresses to 0.0.0.0
Primary WINS Server IP Address
Secondary WINS Server IP Address
Enter the desired IP address of the primary, and optional secondary WINS server you wish the unit to use.
