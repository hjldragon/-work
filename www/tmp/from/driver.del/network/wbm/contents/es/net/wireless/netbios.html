NetBIOS (Inalámbrica)
Visión general
Esta página le permite modificar los parámetros de NetBIOS.
NETBIOS/IP
Activa la comunicación sobre el protocolo de Internet (IP).
Nombre del ordenador
El nombre de sistema es los primeros 15 caracteres del nombre de nodo del dispositivo.
Método del servidor WINS
Para NetBIOS/IP, si se ha seleccionado STATIC, los valores presentes en los campos de servidor WINS se utilizan por el dispositivo. Si está seleccionado AUTO,el dispositivo intentará utilizar DHCP para obtener estos parámetros cuando el servidor se encienda o se restablezca. El método de arranque en la página TCP/IP debe establecerse como Automático o DHCP para activar DHCP. Para desactivar el registro WINS, establezca el método como STATIC y configure las direcciones IP del servidor WINS primario y secundario como 0.0.0.0.
