SMTP
Visión general
Esta página le permite modificar los parámetros del dispositivo.
Ajustes de enviar correo electrónico (SMTP)
Dirección del servidor
Este campo visualiza el nombre de máquina o la dirección IP de un servidor de correo SMTP (Servidor de correo electrónico saliente) en su red. (Por ej. smtp.ejemplo.net -o- 192.0.0.1).
Si está utilizando el nombre DNS del servidor de correo electrónico SMTP, entonces deberá utilizar el FQDN (Fully Qualified Domain Name: Nombre Completo de Dominio) como smtp.ejemplo.net.
Puerto
Especifica el número de puerto de destino del servidor SMTP al que se conectará.
Método de autenticación de servidor
Especifique el método de autentificación para el servidor SMTP. Este valor determina qué método se utiliza para autentificar el usuario en el servidor SMTP.
Ninguno: No se autentifica en el servidor SMTP.
SMTP-AUTH: SMTP-AUTH utiliza el método de Autentificación SMTP. Si se utiliza este método, es necesario definir los detalles de la cuenta SMTP correctamente.
Nombre de cuenta SMTP-AUTH
Contraseña de cuenta SMTP-AUTH
Especifique un nombre y una contraseña que se validarán en el servidor SMTP. Si la contraseña no es necesaria, introduzca un espacio en blanco.
Dirección de correo electrónico del dispositivo
Introduzca la dirección de correo electrónico del dispositivo. Si la dirección del administrador no se ha introducido correctamente, el correo de notificación se enviará al dispositivo. Si no establece una dirección de correo electrónico del dispositivo, el correo de notificación no se enviará al administrador.
