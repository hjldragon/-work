IPv6 (Cableada)
Visión general
Esta página le permite modificar los parámetros de Ipv6.
De forma predeterminada, IPv6 está desactivado. Para activarlo, haga clic en la opción "Activar".
Dar prioridad a la dirección IPv6
Cuando se selecciona esta opción, las direcciones IPv6 tienen prioridad sobre las direcciones IPv4.
Dirección IPv6 estática
Introduzca la dirección IPv6 que desea asignar al dispositivo.
Activar esta dirección
Para utilizar la dirección IPv6 que se ha introducido manualmente, seleccione la opción "Activar esta dirección".
Dirección primaria IP del servidor DNS
Dirección secundaria IP del servidor DNS
Introduzca la dirección IP que desee del servidor DNS primario y secundario opcional.
Lista de direcciones IPv6
La lista de las direcciones IPv6 asignadas a esta máquina.
