Tipo de error
Puede decidir el momento en que se enviará un correo electrónico para un error en particular configurando las opciones "xx minutos" y "xx veces". Se enviará un correo electrónico "xx minutos" después de que se produzca el error seleccionado. Se enviará un correo electrónico cuando el error seleccionado se produzca "xx veces".
