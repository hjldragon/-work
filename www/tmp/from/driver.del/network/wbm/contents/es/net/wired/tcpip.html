TCP/IP (Cableada)
Visión general
Esta página le permite modificar los parámetros de TCP/IP.
NOTA :
Si desactiva el protocolo IP (Protocolo de Internet), no podrá configurar el dispositivo en la web.
Dirección IP
Máscara de subred
Escriba la configuración de la dirección IP que desee asignar al dispositivo (pregunte al administrador de su red qué dirección IP tiene que utilizar).
Puerta de acceso
Si usa una puerta de acceso (enrutador), escriba aquí su dirección (pregunte al administrador de su red qué dirección de puerta de acceso tiene que utilizar).
Método de arranque
Si es STATIC, los valores presentes en los campos Dirección IP, Máscara de subred y Puerta de acceso se utilizarán por el dispositivo. Si no es así, este campo indica el protocolo de red que se utilizará para obtener estos parámetros cuando el servidor se encienda o se restablezca. Si activa el protocolo APIPA, el dispositivo utilizará una dirección IP generada por APIPA si no hay disponible un servidor DHCP o de otro tipo capaz de asignar direcciones IP de forma automática.
