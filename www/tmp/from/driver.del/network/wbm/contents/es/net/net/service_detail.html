Servicio
Visión general
Esta página le permite modificar los parámetros asociados con el servicio que haya seleccionado en la lista de servicios.
Nombre del servicio
Especifica el nombre del servicio. Normalmente, no es necesario modificar este campo.
Puerto de servicio
Especifica el puerto físico asociado con el servicio.
Protocolos
Estas casillas de verificación determinan los protocolos de red activados para este servicio.
Filtro
Especifica el filtro de datos, si lo hay, para este servicio.
0 indica que no hay filtro, los datos pasan sin modificar.
1 indica una sustitución de texto, CRLF por LF de forma predeterminada.
2 indica un trabajo AppleTalk. El filtrado depende del hardware y las opciones seleccionadas.
3 convierte el texto normal a PostScript.
4 convierte los datos de salida en PostScript binario.
Cadenas de control
Especifica las cadenas de datos que se enviarán al principio o al final de un trabajo.
Puerto TCP simple
Si el valor no es cero, define un puerto TCP que se utilizará para la conexión con este servicio.
Bidireccional
Si se marca esta opción, el servicio devolverá datos desde el dispositivo a la red. Normalmente no es necesario cambiar esta opción. En el caso de que disponga de un dispositivo PostScript que no sea bidireccional, puede desactivar la admisión bidireccional para activar el spoofing AppleTalk.
